    acts_as_cerializable(options = {}) click to toggle source

    `acts_as_cerializable` is used to declare that a model uses Cerializable for serialization.

    Unless a module is specified via the serialize_with option, the serializer will attempt to include a module based on the model's name.

    For example, calling Comment.acts_as_cerializable without a serialize_with option will cause Cerializable to look for a CommentSerializer module when setting up the Comment model's serializer.

    Calling Comment.acts_as_cerializable serialize_with: MySerializer will cause Cerializable to look for a MySerializer module when setting up the Comment model's serializer.

def acts_as_cerializable(options = {})
  Cerializable.setup(options.merge(base: self))
